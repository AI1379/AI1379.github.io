洛谷P1216八皇后题解

本文最后更新于：8 个月前

题目链接

极为经典的DFS练习题

首先我们需要一个数组vis[15][15]用来标记棋盘上哪些点不能放皇后。”显而易见”的这是个bool数组(加引号的原因后面再说)

其次我们还需要一个数组ans[15]来存储当前答案，然后再用sum表示解的总个数

然后读入，初始化(vis数组全部设为0)，然后就开始DFS:

void dfs(int dep) //dep表示当前所在行
{
	int i, j;
	if (dep == n)
	{
		sum++;
		if (sum <= 3)//输出
		{
			for (i = 1; i <= n; i++)
			{
				cout << ans[i];
			}
		}
		memset(ans, 0, sizeof(ans));
		memset(vis, 0, sizeof(vis));
		return;//回溯
	}
	for (i = 1; i <= n; i++)
	{
		if (!vis[dep][i])
		{
			ans[dep] = i;
			//标记
			for (j = dep; j <= n; j++)
				vis[j][i] = true;
			for (j = 1; j <= n - dep && j <= i - 1; j++)
				vis[dep + j][i - j] = true;
			for (j = 1; j <= n - dep && j <= n - i; j++)
				vis[dep + j][i + j] = true;
			dfs(dep + 1);
		}
	}
	return;
}

别抄这段代码！
它是错的！！！

那我们来看看哪里错了。

首先我们发现第一个if语句应该放到循环里面的if语句里面并且去掉return;，否则可能会漏解。

其次两个memset也是不对的。因为我们答案在存入ans数组的时候是会覆盖原有数据的，所以memset(ans,0,sizeof(ans));一句可以直接删去，回溯查找下一个分支的时候直接覆盖存储，但把vis数组恢复到当前状态的前一个状态有点麻烦。

首先很容易就可以想到，可以将memset(vis,0,sizeof(vis))一句改为如下代码:

for (j = dep; j <= n; j++)
	vis[j][i] = false;
for (j = 1; j <= n - dep && j <= i - 1; j++)
	vis[dep + j][i - j] = false;
for (j = 1; j <= n - dep && j <= n - i; j++)
	vis[dep + j][i + j] = false;

好像可以！

但如果细想的话还是不行，因为当你这么赋值了之后，一些原来的数据就被破坏掉了。

就比如说我们在6*6的棋盘中，点(1,3)上和点(5,2)上各有一个皇后。那么我们看一看(6,3)这个点：显而易见的在(1,3)的皇后可以威胁到(6,3)这个点，但当我们回溯到(5,2)时就会把(6,3)的这个值设为false，即这个点是安全的，但实际上不是。

那怎么办呢？我们可以把vis定义为一个三维bool数组来存储当前情况下每多放一个皇后后棋盘上的情况，但这样实现起来太复杂了而且容易错。

所以怎么解决这个问题呢？问题出在之前的”显而易见”上。

如果把vis定义为一个int数组来表示能威胁到这个点的皇后的个数，那么这事就好办了：搜索时每多放一个皇后把新增的格子++，回溯的时候- -就行了。同时因为我们是一行一行往下搜索的，所以横排的可以不用处理，其他三个方向只要从当前位置往下走就行了，可以略微优化一下。

因此完整代码如下

#include <bits/stdc++.h>
using namespace std;
int ans[15];
int vis[15][15];
int n;
long long sum = 0;
void dfs(int dep)
{
	int i, j;
	for (i = 1; i <= n; i++)
	{
		if (!vis[dep][i])
		{
			ans[dep] = i;
			for (j = dep; j <= n; j++)
				vis[j][i]++;
			for (j = 1; j <= n - dep && j <= i - 1; j++)
				vis[dep + j][i - j]++;
			for (j = 1; j <= n - dep && j <= n - i; j++)
				vis[dep + j][i + j]++;
			if (dep == n)
			{
				sum++;
				if (sum <= 3)
				{
					for (j = 1; j <= n; j++)
					{
						cout << ans[j] << ' ';
					}
					cout << endl;
				}
			}
			if (dep < n)
				dfs(dep + 1);
			for (j = dep; j <= n; j++)
				vis[j][i]--;
			for (j = 1; j <= n - dep && j <= i - 1; j++)
				vis[dep + j][i - j]--;
			for (j = 1; j <= n - dep && j <= n - i; j++)
				vis[dep + j][i + j]--;
		}
	}
	return;
}
int main()
{
	cin >> n;
	memset(ans, 0, sizeof(ans));
	memset(vis, 0, sizeof(vis));
	dfs(1);
	cout << sum;
	return 0;
}
